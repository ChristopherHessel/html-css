Tags

  • p = paragrafo
  • br = quebrar linhas
  • ul = listas não ordenadas
  • ol = listas Ordenadas
  • li = item de alguma lista
  • h1 = Titulo
  • h2 = subtitulo
  • h3,h4,h5 e h6 são subs um dos outros.
  • strong = deixar em negrito
  • i = deixar em italico
  • link = linkar algo para o documento
  • img = colocar imagens
  • source = modificar videos,imagens e audios

      Atributos

    • Control = mostra os controles em audios/videos
    • autoplay = inicia a reprodução assim que o site for carregado
    • loop = faz com que o audio/video fique repetindo

  • picture = tag para imagens
  • audio = tag para audios
  • video = tag para videos
  • pre = mostra todo o codigo do jeito q esta no VScode
  • ins = inserir texto desse jeito
  • del = deletar texto desse jeito: Deletado
  • mark = marcar textos assim Texto marcado
  • dl = listas de Definições
  • q = citação simples EX:(Exemplo)
  • blockquote = citaçao completa EX:(
    Exemplo
  • abbr = abreviação de palavras EX:(abbr)
  • small = Deixar texto menos EX:(Exemplo)
  • code = mostra o texto em fonte de codigo EX: (Exemplo)
  • em = Enfase na frase EX:(este é o Exemplo)
  • div = é como criar uma caixa para organizar as coisas
  • a = levar algo para outro documento ou lugar

Emojis e simbolos

  • ♦ Simbolo EX1: &(Nome do simbolo) obs: só funcionam para alguns simbolos.

  • ✈ Simbolo EX2: &#(Código do Simbolo)
