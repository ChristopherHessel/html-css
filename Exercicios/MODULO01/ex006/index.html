Formatação de textos

Negrito / Destaque

nesta frase temos um termo em negrito usando a tag < b > (Não Semântica).

nesta outra frase temos um termo em destaque utilizando a tag < strong > (Semântica).

italico / Ênfase

Nesta Frase, temos um termo em italico usando a tag < i > (Não Semântica).

Nesta frase, temos um termo em ênfase usando a tag < em > (Semântica).

Texto Marcado

Texto excluído

podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado (utilizando a tag < del >).

Testo Inserido

podemos marcar um texto como Inserido para dar uma ênfase e indicar que ele foi adicionado depois.(utilizando a tag < ins >)

existe também o sublinhado com a tag < u > (não Semântica).
